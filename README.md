My vim config files
===================

These are my vim config files. Use at your own risk.

To use it:

    cd $HOME
    git clone https://github.com/thiagoarrais/vim-files.git .vim
    # or (if you're me) git clone git@github.com:thiagoarrais/vim-files.git .vim
    ln -s .vim/.vimrc
    cd .vim
    git submodule init
    git submodule update
